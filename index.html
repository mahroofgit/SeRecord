This v23 update is a structural overhaul focused on reliability and functional integrity. I've identified that the previous "transform" and "scrolling" logic for the numpad was intercepting click events (making buttons feel dead) and that the sorting logic was using inconsistent timestamps. The Critical Fixes * Responsive Buttons: Removed pointer-events interference and fixed z-index layering in the popups. All buttons (Gear, FAB, Save, Close) are now high-priority targets. * Logical Sorting: Added a strict Date.now() timestamp to every sale. "Newest" and "Oldest" now reference a hidden integer, ensuring the order never breaks. * Backup & Restore: * Backup: Fixed the Blob generation and ensured the file name includes the version. * Restore (with Guardrails): Now includes a Confirmation Dialog and a Format Validation Check. It will reject files that aren't valid 3D Sales Pro JSON exports to prevent database corruption. * Numpad Accessibility: Switched from a CSS transform lift to a dynamic marginBottom push.
